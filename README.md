## IA Generativa:

Con el fin de entender al máximo el funcionamiento de Flutter he priorizado usar el minimo de IA Generativa posible. Limitando su uso a los siguientes casos:

1. **Autocompletado [GitHub Copilot]**: Uso del autocompletado de Visual Studio Code con el fin de escribir mas rápido o para saber bien bien como se introduce ciertos formatos de datos.
2. **Lectura de codigo [ChatGPT]**: Explicaciones más detalladas y claras sobre algunos log de errores obtenidos y partes del codigo.
3. **Soporte de escritura [ChatGPT]**: Ayuda para la familarización con el lenguaje Dart y soporte para picar codigo en dicho lenguaje.
4. **Identación [ChatGPT]**: Soporte para redactar de forma precisa y saber que puntos son los convenientes para modificar el codigo.

El uso de IA ha sido evitado para cualquier otra situación.
